Руководство по CSS

Основы CSS

CSS (Cascading Style Sheets) - это язык стилей, определяющий отображение HTML-документов.

/* Изменение цвета текста и фона */
body {
    font-family: Arial, sans-serif;
    background-color: #f8f9fa;
    color: #333;
}

/* Стили для заголовка */
h1 {
    color: #007bff;
    text-align: center;
    margin-bottom: 1rem;
}

/* Стили для кнопок */
.button {
    background-color: #28a745;
    color: white;
    padding: 0.5rem 1rem;
    border-radius: 0.25rem;
}

Flexbox

Flexbox - это современная модель макета для создания гибких структур.

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
}

.item {
    flex: 1;
    margin: 0 10px;
}

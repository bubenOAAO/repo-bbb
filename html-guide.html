Основы HTML

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц.

<!DOCTYPE html>
<html>
<head>
    <title>Моя страница</title>
</head>
<body>
    <h1>Привет, мир!</h1>
    <p>Мой первый HTML-документ</p>
</body>
</html>
